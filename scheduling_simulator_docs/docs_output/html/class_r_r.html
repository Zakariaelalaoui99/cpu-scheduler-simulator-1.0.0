Detailed Description

Round-Robin scheduler with fixed quantum (preemptive).

The Round-Robin scheduler maintains a FIFO ready queue of processes. Each process that arrives is enqueued in arrival order, then scheduled in cyclic fashion:

  • Runs for at most quantum time units.
  • If the process still has remaining burst time after its slice, it is re-enqueued at the back of the ready queue.
  • If completed, its completion/turnaround/waiting metrics are finalized.

This scheduler ensures fairness by time-slicing the CPU among all ready processes.
