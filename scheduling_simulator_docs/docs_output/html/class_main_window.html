Detailed Description

Top-level Qt window for the Scheduling Simulator.

Provides the graphical user interface for creating processes/threads, selecting scheduling algorithms (FCFS, SJF, RR, RRP), choosing the execution mode (process-only, kernel-level threads, user-level threads), and configuring simulation parameters such as context switch time and quantum.

Interaction model:

  • Emits signals (e.g., processCreated, algorithmSelected, runRequested) based on user actions.
  • The Controller listens to these signals, runs the Scheduler, and returns results.
  • Slots such as showProcesses, displayResult, and displayGantt update tables and the Gantt chart view.
