Detailed Description

Schedulable thread owned by a Process.

Each Thread encapsulates scheduling parameters such as arrival time, burst duration, and priority. Depending on the Scheduler mode:

  • In ThreadKernel mode, threads are treated as independent schedulable units, flattened into pseudo-processes.
  • In ThreadUser mode, threads are multiplexed in user space within the time slice allocated to their parent process.

Threads also track results (completion, turnaround, waiting time) after scheduling is complete.
